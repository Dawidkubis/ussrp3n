# UssrP3n

```
ussrp3n 0.1.0

USAGE:
    ussrp3n [OPTIONS] <command> [ARGS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -t, --threads <threads>    number of threads to run on [default: 1]

ARGS:
    <command>      the file with curl command
    <users>        users list
    <passwords>    password dictionary
```
